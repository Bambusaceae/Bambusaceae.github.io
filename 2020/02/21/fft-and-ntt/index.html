# 快速傅里叶变换（FFT）

上面讲了一大堆什么单位根，有什么用呢？没错！我们要把它们带入多项式里去。

前面说过，一个 nnn 次多项式 F(x)F(x)F(x) 可以由 n+1n+1n+1 个不同的点 (xi,F(xi))(x_i,F(x_i))(xi​,F(xi​)) 确定。我们这里令nnn为 F(x)F(x)F(x) 的次数 +1+1+1。

但显然，仅仅是简单的带入和朴素算法没区别。但利用单位根的性质，我们可以使用分治进行优化。

不妨把一个多项式 F(x)F(x)F(x) 按照奇偶性分成两组：

F(x)=f0x0+f1x1+f2x2+…=(f0x0+f2x2+f4x4+…)+(f1x1+f3x3+f5x5+…)=(f0x0+f2x2+f4x4+…)+x(f1x0+f3x2+f5x4+…)\begin{aligned}F(x)&=f_0x^0+f_1x^1+f_2x^2+\dots\\&=(f_0x^0+f_2x^2+f_4x^4+\dots)+(f_1x^1+f_3x^3+f_5x^5+\dots)\\&=(f_0x^0+f_2x^2+f_4x^4+\dots)+x(f_1x^0+f_3x^2+f_5x^4+\dots)\end{aligned}F(x)​=f0​x0+f1​x1+f2​x2+…=(f0​x0+f2​x2+f4​x4+…)+(f1​x1+f3​x3+f5​x5+…)=(f0​x0+f2​x2+f4​x4+…)+x(f1​x0+f3​x2+f5​x4+…)​

# 快速傅里叶逆变换

转换成点至表示法运算完之后还需要重新转换成系数表示法。

设 gi=F(ωni)g_i=F(\omega_n^i)gi​=F(ωni​)，G(x)=∑i=0ngixiG(x)=\sum_{i=0}^ng_ix^iG(x)=∑i=0n​gi​xi

于是我们可以考虑上网搜题解。

网上的题解显示先带一个 ωn−k\omega_n^{-k}ωn−k​ 。然后直觉告诉我们这是对的。

G(ωn−k)=∑i=0n−1giωn−ki=∑i=0n−1(∑j=0n−1fjωnij)ωn−ki=∑i=0n−1∑j=0n−1fjωnijωn−ki=∑j=0n−1fj∑i=0n−1ωn(j−k)i\begin{aligned}G(\omega_n^{-k})&=\sum_{i=0}^{n-1}g_i\omega_n^{-ki}\\&=\sum_{i=0}^{n-1}(\sum_{j=0}^{n-1}f_j\omega_n^{ij})\omega_n^{-ki}\\&=\sum_{i=0}^{n-1}\sum_{j=0}^{n-1}f_j\omega_n^{ij}\omega_n^{-ki}\\&=\sum_{j=0}^{n-1}f_j\sum_{i=0}^{n-1}\omega_n^{(j-k)i}\end{aligned}G(ωn−k​)​=i=0∑n−1​gi​ωn−ki​=i=0∑n−1​(j=0∑n−1​fj​ωnij​)ωn−ki​=i=0∑n−1​j=0∑n−1​fj​ωnij​ωn−ki​=j=0∑n−1​fj​i=0∑n−1​ωn(j−k)i​​

我们先对 ∑i=0n−1ωn(j−k)i\sum_{i=0}\limits^{n-1}\omega_n^{(j-k)i}i=0∑n−1​ωn(j−k)i​ 这个式子下手。

分两种情况讨论：

  1. 当 j=kj=kj=k 时，显然这个式子的结果是 nnn。
  1. 当 j≠kj\neq kj=k 时，我们就用到了我们的等比数列求和大法。先看这个等式：
∑i=0n−1ωn(j−k)i=∑i=0n−1ωn(j−k)i\sum_{i=0}^{n-1}\omega_n^{(j-k)i}=\sum_{i=0}^{n-1}\omega_n^{(j-k)i}∑i=0n−1​ωn(j−k)i​=∑i=0n−1​ωn(j−k)i​

它显然成立。我们在这个等式两边同时乘上 ωnj−k\omega_n^{j-k}ωnj−k​，得到：

ωnj−k∑i=0n−1ωn(j−k)i=∑i=0n−1ωn(j−k)(i+1)\omega_n^{j-k}\sum_{i=0}^{n-1}\omega_n^{(j-k)i}=\sum_{i=0}^{n-1}\omega_n^{(j-k)(i+1)}ωnj−k​∑i=0n−1​ωn(j−k)i​=∑i=0n−1​ωn(j−k)(i+1)​

两等式相减，得

(ωn−1j−k−1)∑i=0n−1ωn(j−k)i=1−ωnn(\omega_{n-1}^{j-k}-1)\sum_{i=0}^{n-1}\omega_n^{(j-k)i}=1-\omega_n^{n}(ωn−1j−k​−1)∑i=0n−1​ωn(j−k)i​=1−ωnn​

想想单位根的某个性质。

(ωn−1j−k−1)∑i=0n−1ωn(j−k)i=0⇒∑i=0n−1ωn(j−k)i=0\begin{aligned}&\ \ \ \ \ \ (\omega_{n-1}^{j-k}-1)\sum_{i=0}^{n-1}\omega_n^{(j-k)i}&=&0\\&\Rightarrow\sum_{i=0}^{n-1}\omega_n^{(j-k)i}&=&0\end{aligned}​(ωn−1j−k​−1)i=0∑n−1​ωn(j−k)i​⇒i=0∑n−1​ωn(j−k)i​​==​00​

G(ωn−k)=∑j=0n−1fj∑i=0n−1ωn(j−k)i=nfk⇒fk=G(ωn−k)n\begin{aligned}&\ \ \ \ \ \ G(\omega_n^{-k})&=&\sum_{j=0}^{n-1}f_j\sum_{i=0}^{n-1}\omega_n^{(j-k)i}\\&&=&nf_k\\&\Rightarrow f_k&=&\frac{G(\omega_n^{-k})}{n}\end{aligned}​G(ωn−k​)⇒fk​​===​j=0∑n−1​fj​i=0∑n−1​ωn(j−k)i​nfk​nG(ωn−k​)​​
